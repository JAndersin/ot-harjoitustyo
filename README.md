# Province Data Editor

Sovellus on työkalu helpottamaan bittikarttapohjaisen strategiapelin JSON-tiedostoihin tallennettujen olioiden tarkastelua ja muokkausta. Nykyisessä muodossaan sillä voidaan valita hiiren vasemmalla painikkeella mikä tahansa alue kartalta ja tutkia sen tietoja ja tarvittaessa muokata niitä ja tallentaa ne JSON-tiedostoon. Ei sisällä alustusta, joten oletuksena mikään provinssi ei sisällä mitään tietoa vaan käyttäjän tulee itse lisätä ja tallentaa se, joka onnistuu jo tällä työkalulla.
